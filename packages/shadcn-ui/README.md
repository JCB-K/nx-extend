**Nx plugin for working with [shadcn/ui](https://ui.shadcn.com/)**.

## Setup

### Install

```sh
npm install -D @nx-extend/shadcn-ui
nx g @nx-extend/shadcn-ui:init
```

## Usage

### Add

```sh
nx g @nx-extend/shadcn-ui:add button
```
